### To compile and run the code

```
$ mvn clean install
```

#### For Windows:
- Bootstrap node:
    ```
    $ java -jar .\target\UserInterface-1.0-SNAPSHOT.jar bootstrap [ip] [port]
    ```
- Normal node:
    ```
    $ java -jar .\target\UserInterface-1.0-SNAPSHOT.jar normal [ip] [port]
    ```

#### For Mac or Linux:
- Bootstrap node: 
    ```  
    $ java -jar ./target/UserInterface-1.0-SNAPSHOT.jar bootstrap [ip] [port]
    ```
- Normal node:
    ```
    $ java -jar ./target/UserInterface-1.0-SNAPSHOT.jar normal [ip] [port]
    ```
